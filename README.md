HTML5本地視頻播放器
===

這所謂"HTML5本地視頻（視訊）播放器"，說白了就是一HTML頁面，目的是使得使用瀏覽器播放本地視頻更方便。

提供功能：

*基本的視頻播放

*洗腦循環

*顯示已播放百分比和剩餘時間

*固定大小（縮放）

*用HTML5本地存儲保存設定的大小，並可清除以使用原始大小。


視頻地址（路徑）格式:
<pre>
WINdos:
file:///盤符:/path/to/file.xxx
UNIX-LIKE:
file:///path/to/file.xxx
</pre>

支持格式取決於瀏覽器。
